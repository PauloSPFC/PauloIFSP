GitHub

Passo a Passo para a administração de um repositório

Criando sua conta

  1. Para criar sua conta, acesse GitHub.com
  2. No canto superior direito, clique em Sign Up
  3. Insira seus dados e clique em Create an Account
  4. Depois, selecione o seu plano GitHub e clique em Continue
  5. Informe sua experiência com programação e clique em Submit

Criando seu Repositório

  1. No canto superior direito, clique em + e então, em New repository.
  2. Nomeie Seu Repositório.
  3. Escreva uma descrição curta.
  4. Selecione Initialize this repository with a README.
  5. Selecione Create Repository.

Salvando seus arquivos no Repositório

  1. Efetue o Download do GIT, acessando GitHub Download
  2. Após a instalação, o programa estará disponível na Área de Trabalho.
  3. Crie uma Pasta e adicione um arquivo de texto na mesma, com qualquer texto inserido
  4. Depois, clique com o botão direito na pasta e selecione Git GUI here
  5. No programa, selecione Edit >> Options
  6. Na aba Global, insira seu nome de usuário E-mail do GitHub
  7. Clique em Stage Changed
